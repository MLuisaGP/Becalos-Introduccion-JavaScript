Document Objet Model

Interfaz que representa al HTML como un arbol de nodos.
Sirve para manipular elementos html mediante javascript utilizando metodos, podemos cambiar atribulos, estilos, etc.

Nodo raiz -> representa todo el documento DOCTYPE
Nodo HTML -> Nodo principal
Nodo head y body ->hijos de principal
Nodos elemento ->Representa todos los elementos (p, meta, h1 ,etc)
Nodos textos-> Representa el contenido
Nodos Atributos -> Represeta los atributos como id, class src etc.
Nodos de comentarios
